Demonstrations of udpsend, the Linux eBPF/bcc version.


This tool traces UDP sendmsg() calls, showing the process name, PID, UID, IP Version, source and destination IP addresses and ports.:

# ./udpsend.py
Tracing UDP sendmsg ... Hit Ctrl-C to end
PID     UID     COMM         IP SADDR            SPORT DADDR            DPORT
83694   101     isc-net-0000 4  192.168.170.129  36350 192.168.170.2    53
83696   1000    ping         4  192.168.170.129  39403 192.168.170.2    53
83696   1000    ping         4  192.168.170.129  39403 192.168.170.2    53
83696   1000    ping         4  192.168.170.129  33407 192.168.170.2    53
83700   1000    curl         4  192.168.170.129  48348 192.168.170.2    53
83700   1000    curl         4  192.168.170.129  48348 192.168.170.2    53

USAGE message:

# ./udpsend.py -h
usage: udpsend.py [-h] [-T] [-t]

Trace UDP sends

options:
  -h, --help       show this help message and exit
  -T, --time       include time column on output (HH:MM:SS)
  -t, --timestamp  include seconds since trace start

examples:
    ./udpsend           # trace all UDP sendmsg()s
    ./udpsend -t        # include seconds since trace start
    ./udpsend -T        # include wall-clock time (HH:MM:SS)
